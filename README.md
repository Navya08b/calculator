# calculator

Glassmorphism Calculator

A modern and responsive calculator built using HTML, CSS, and JavaScript. The project features a clean glassmorphism-inspired user interface with a colorful gradient background and performs basic arithmetic operations such as addition, subtraction, multiplication, and division. It demonstrates fundamental front-end development concepts, including DOM manipulation, event handling, responsive layout, and interactive UI design.

Features

- Modern glassmorphism UI
- Responsive design
- Basic arithmetic operations
- Interactive button hover effects
- Built with pure HTML, CSS, and JavaScript (no frameworks)

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
